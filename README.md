# Uzzal-Renderer-PyPi-Package
```bash
This repository contains the source code and documentation for the Uzzal Renderer Python package, which is available on PyPi. The Inception Renderer is a powerful tool for rendering any youtube videos and any web page over the Jupyter/Colab Notebook.
```

# Installation:

### Create a conda environment
```bash
conda create -n renderer python=3.8 -y

conda activate renderer

pip install -r requirements_dev.txt

```
